# Gramadoc

Gramadoc is a modern writing environment with real-time grammar and style analysis.

## Features

- **Rich Text Editing:** Full-featured editor with support for headings, lists, bold, italic, and more.
- **Real-time Analysis:** Instant feedback on grammar, style, and spelling.
- **Multiple Themes:** Choose from various light and dark themes to suit your writing style.
- **Privacy First:** Your documents are saved locally in your browser.
- **PWA Support:** Install Gramadoc as a desktop or mobile application for an offline-ready experience.

## Getting Started

1. Clone the repository.
2. Install dependencies: `pnpm install`
3. Start the development server: `pnpm dev`
4. Build for production: `pnpm build`

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Vanilla CSS with modern CSS variables
- **Analysis Engine:** Custom rules and grammar analysis package
- **Persistence:** LocalStorage with PWA capabilities
